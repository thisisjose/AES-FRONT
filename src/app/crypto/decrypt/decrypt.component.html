AES-256-CBC

AES-256-CBC es un algoritmo de cifrado simétrico que convierte un texto en un formato ilegible (texto encriptado) usando una clave secreta de 256 bits. Para desencriptar, se utiliza la misma clave secreta y el IV (vector de inicialización) correcto. Ingresa el texto encriptado y el IV para obtener el texto original.
